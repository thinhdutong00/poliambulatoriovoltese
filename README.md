# Poliambulatorio Voltese

Sito statico iniziale per Poliambulatorio Voltese.

## Deploy su Vercel

Configurazione consigliata:

- Framework preset: `Other`
- Root directory: `./`
- Build command: lasciare vuoto
- Output directory: lasciare vuoto oppure `./`

Il sito non richiede dipendenze o passaggi di build.
